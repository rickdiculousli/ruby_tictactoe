# Tic Tac Toe

A tic tac toe game from the console.

## How to run

Change directory into project, then run Ruby intepretor on the Game.rb file

```
$ cd ./ruby_tictactoe/lib
$ ruby Game.rb
```

### Exiting

Simply input CTRL + C to exit from the game.
